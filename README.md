# Alajangi Vidya Sagar - Portfolio

A modern, high-performance portfolio showcasing full-stack development projects, AI integrations, and blockchain applications.

## 🚀 Projects

- **CertGuard (Blockchain)**: Digital certificate platform with AWS S3 integration.
- **MedConnect (AI Healthcare)**: Telehealth platform with Gemini AI triage and Agora video calls.
- **FoodFrenzy (MERN/Spring)**: Full-stack restaurant management system.
- **ARC Trust (Non-Profit)**: Community-driven platform for non-profit operations.

## 🛠️ Tech Stack

- **Frontend**: React.js, Tailwind CSS, GSAP (Animations)
- **Backend**: Node.js, Spring Boot, Python
- **Database**: MongoDB, MySQL
- **Cloud/AI**: AWS S3, Gemini API, Agora SDK, Blockchain

## 📦 Getting Started

### Prerequisites
- Node.js (v18+)
- npm

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/VidyasagarAlajangi/portfolio.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```
